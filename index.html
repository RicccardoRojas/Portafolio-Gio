GIOVANNI RICARDO ROJAS GARCIA

Soy un ingeniero en Sistemas Computacionales recién egresado y actualmente en proceso de titulación. Me apasiona el aprendizaje continuo y destaco por mis habilidades en programación, con experiencia en el desarrollo de soluciones tecnológicas eficientes. Durante mis prácticas profesionales, acumulé medio año de experiencia en el desarrollo de un sistema cliente-servidor que transformó la gestión empresarial, eliminando la dependencia de formatos físicos y digitalizando completamente los procesos administrativos. Este proyecto incluyó el diseño y desarrollo de una API en .NET Core Web API, que se conecta a una base de datos MySQL mediante consultas optimizadas para gestionar y almacenar información empresarial. Utilicé JSON Web Tokens (JWT) para garantizar la seguridad de los datos, permitiendo acceso únicamente a usuarios autenticados. La API devuelve los datos al cliente, una aplicación desarrollada en C#, la cual interpreta y presenta la información al usuario final de manera clara y funcional. Este enfoque permitió implementar una solución robusta, segura y escalable para la empresa.
