ALIMENTOS DESHIDRATADOS DE JALISCO

Lagos de Moreno, Jalisco

Alimentos Deshidratados de Jalisco, localizada en la ciudad de Lagos de Moreno, Jalisco, es una empresa pionera en el proceso de deshidratación de leche, especializada en la conversión de la leche líquida en suero. Mediante un avanzado proceso de secado, transformamos la leche en polvo, lo cual no solo mejora su conservación, sino que también facilita su transporte y almacenamiento a larga distancia. Este suero de leche es un ingrediente esencial que se utiliza ampliamente en la industria alimentaria, especialmente para la producción de productos lácteos procesados, suplementos nutricionales y diversas bebidas.

DESARROLLO DEL PROYECTO

BASE DE DATOS

En el desarrollo del proyecto de base de datos, se utilizó MySQL como el sistema de gestión de bases de datos para almacenar y gestionar la información de manera eficiente. A través de MySQL Workbench, se diseñó el diagrama relacional, que permitió visualizar claramente las relaciones entre las distintas tablas del sistema. Además, se implementaron procedimientos almacenados para realizar operaciones cruciales, como insertar, actualizar, eliminar y consultar datos, optimizando las consultas y garantizando la integridad de los datos. Este enfoque facilitó la administración de la base de datos y mejoró el rendimiento general del sistema.

API

ASP.NET CORE WEB API
Este proyecto se desarrolló utilizando ASP.NET Core Web API para crear un servicio backend robusto, centrado en la gestión y manipulación de datos en una base de datos. El proyecto comenzó con el diseño de un diagrama relacional para modelar la estructura de las tablas y sus relaciones, lo que permitió una gestión eficiente de la información.
SWAGGER
La API que alimenta este sistema corre sobre un entorno que permite su prueba y documentación de manera interactiva mediante Swagger. Esta herramienta facilita a los desarrolladores explorar y probar los puntos finales de la API, visualizar los parámetros necesarios para cada solicitud y realizar pruebas directamente desde una interfaz gráfica. Swagger hace que la documentación de la API sea clara y accesible, lo que mejora tanto el desarrollo como la integración de servicios.
Json Web Token
Una parte fundamental del proyecto es la seguridad, que se maneja utilizando JWT (JSON Web Tokens). Esta tecnología asegura que solo los usuarios autenticados y autorizados puedan acceder a los puntos finales de la API. Al usar JWT, cada solicitud realizada a la API está acompañada de un token de acceso, que es verificado para confirmar la identidad del usuario y garantizar que solo aquellos con las credenciales adecuadas puedan ejecutar las operaciones disponibles.

SERVIDOR

IIS

La API fue implementada y configurada en un servidor IIS (Internet Information Services), lo que permite su alojamiento y acceso a través de un entorno seguro y estable. IIS proporciona una plataforma robusta para ejecutar aplicaciones web, lo que garantiza un rendimiento óptimo y la capacidad de manejar múltiples solicitudes simultáneamente. Este despliegue facilita la integración con otros servicios, asegura una administración centralizada y permite configuraciones específicas para mejorar la seguridad y el control del acceso a la API.

IIS

IIS (Internet Information Services) se utiliza para alojar aplicaciones ASP.NET Core Web API debido a su robustez, escalabilidad y capacidad para manejar múltiples solicitudes simultáneamente. Ofrece una gestión centralizada de seguridad, autenticación y autorización, y soporte para HTTPS. Además, su integración con ASP.NET Core garantiza un rendimiento óptimo, permitiendo la administración eficiente de aplicaciones web en entornos empresariales.
